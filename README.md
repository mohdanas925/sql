# SQL Repository

This repository contains SQL queries and practice problems created while learning SQL and Database Management concepts.

1. Database Basics
2. DDL Commands
   - CREATE
   - ALTER
   - DROP
   - TRUNCATE
3. DML Commands
   - INSERT
   - UPDATE
   - DELETE
4. DQL Commands
   - SELECT
5. Filtering Data
   - WHERE
   - BETWEEN
   - IN
   - LIKE
6. Sorting and Grouping
   - ORDER BY
   - GROUP BY
   - HAVING
7. Joins
   - INNER JOIN
   - LEFT JOIN
   - RIGHT JOIN
   - FULL JOIN
   - SELF JOIN
   - CROSS JOIN
8. Subqueries
9. Aggregate Functions
10. Constraints

- Improve data analysis skills
